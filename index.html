Tema 1: Completa la Tabla

Concepto Descripción Concepto Descripción
Tabla en HTML Es una estructura para organizar datos en filas y columnas. Formulario en HTML Se usa para recopilar datos del usuario.
Etiqueta <table> Define una tabla. Etiqueta <form> Define un formulario para la entrada de datos.
Atributos de <table> Incluye atributos como border, cellpadding, cellspacing. Etiqueta <input> Define un campo de entrada en un formulario.
Etiqueta <tr> Define una fila en una tabla. Etiqueta <label> Define una etiqueta para un control de formulario.
Atributo <tr> No tiene atributos específicos, pero puede incluir clases y estilos. Etiqueta <select> Define un menú desplegable.
Etiqueta <td> y <th> <td> define una celda de datos, <th> define una celda de encabezado. Etiqueta <textarea> Define un área de texto multilínea en un formulario.
Atributos de <td> y <th> Incluyen colspan y rowspan para combinar celdas. Etiqueta <fieldset> y <legend> <fieldset> agrupa controles de formulario, y <legend> define una leyenda.
Tablas avanzadas Incluyen elementos como <thead>, <tfoot>, y <tbody> para una mejor organización. Etiqueta <meter> Define una entrada de datos en un rango definido.
Etiqueta <thead> Define el encabezado de una tabla. Etiqueta <progress> Representa el progreso de una tarea.
Etiqueta <tfoot> Define el pie de una tabla. Etiqueta <datalist> Define una lista de opciones predefinidas para un campo de entrada.
Etiqueta <tbody> Define el cuerpo de una tabla. Etiqueta <summary> Define un resumen para un elemento <details>.

Tema 2: Elementos que Componen la Estructuración de una Página Web

  • Contenedor (wrapper): Un contenedor que engloba todo el contenido de la página para aplicar estilos o centrar el contenido.
  • Cabecera (header): La parte superior de la página que suele incluir el logotipo, el menú de navegación y otros elementos importantes.
  • Contenido (content): La sección principal de la página donde se presenta la información principal.
  • Menú (menu): Una lista de enlaces de navegación que ayuda a los usuarios a moverse por el sitio web.
  • Pie (footer): La parte inferior de la página que suele contener información de contacto, derechos de autor y enlaces adicionales.
  • Lateral (sidebar): Una sección lateral que puede contener enlaces, widgets, o información adicional relevante.
  • Etiqueta div: Un elemento contenedor que se utiliza para agrupar otros elementos y aplicar estilos.

Tema 3: Estructuración Semántica de una Página Web

  • <header>: Define el encabezado de la página o una sección. Generalmente contiene el título, logotipo y navegación principal.
  • <nav>: Define un conjunto de enlaces de navegación.
  • <main>: Representa el contenido principal del documento, excluyendo el encabezado, pie de página y barras laterales.
  • <article>: Define un contenido independiente que puede ser distribuido y reutilizado, como una entrada de blog o un artículo de noticias.
  • <section>: Define una sección genérica de contenido en el documento, que puede contener encabezados, párrafos y otros elementos.
  • <aside>: Define contenido relacionado con el contenido principal, como una barra lateral o una nota adicional.
  • <footer>: Define el pie de página del documento o una sección. Generalmente contiene información de derechos de autor, enlaces de contacto y otros detalles finales.
  • <h1> a <h6>: Definen los encabezados del contenido, donde <h1> es el más importante y <h6> el menos importante.
  • <p>: Define un párrafo de texto.
  • <figure>: Define contenido visual como imágenes, gráficos o diagramas junto con una leyenda.
  • <figcaption>: Define una leyenda para el contenido dentro de un elemento <figure>.
  • <time>: Define una fecha u hora.
